waiterbrain

a local, on-premises voice ordering assistant for restaurants.

how it works

each server wears a mic. a proximity sensor identifies which table they are near.

speech is transcribed locally, structured into an order, and read back to the server for confirmation.

confirmed orders are sent to the pos system via the toast api.

everything runs on a single machine in the back office. no data leaves the building.

specs

target hardware
rtx 4070 (or equivalent)
concurrent voices
up to 8
pos integration
toast api
data storage
local only, resets nightly
footprint
one machine, runs all day
pilot location
amigos mexican kitchen and tequila bar

waiterbrain is a tool for servers, not a replacement for them.
