superstopper

plugs in via usb-c and controls the timer. sub-10ms from press to response. the button lights up and flashes too.

  • tap to start
  • tap while running to pause
  • double tap to reset
  • triple tap to cycle the goal time
  • hold to cycle precision

firmware

spacebar_firmware.ino

wiring

momentary switch: one leg to pin 2, other leg to GND. INPUT_PULLUP is set in firmware so no external resistor needed.

LED: anode through a 330 ohm resistor to pin 9, cathode to GND.
